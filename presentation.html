CSS `filter()` — қысқаша түсінік

Пайдалану: суреттерге, элементтерге (SVG, canvas, блоктар) визуал әсер қосу.

Негізгі функциялар

  • none — ешқандай эффект жоқ
  • blur(px) — бұлыңғырлау
  • brightness() — жарықтық
  • contrast() — контраст
  • drop-shadow() — көлеңке
  • grayscale() — сұрға айналдыру
  • hue-rotate() — түсті айналдыру
  • invert() — инверсия
  • opacity() — мөлдірлік
  • saturate() — қанықтылық
  • sepia() — сепия
  • url(<filterId>) — SVG фильтрі
Жедел мысал 
/* apply multiple */
.photo { filter: blur(2px) saturate(140%); }

/* transition for smooth preview */
img { transition: filter .3s ease, transform .35s ease }
